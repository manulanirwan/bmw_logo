# BMW Logo Generator using Python Turtle

A simple and clean Python script that recreates the iconic BMW logo from scratch using the built-in `turtle` graphics library. 

## 🚀 Features
* Draws the complete outer black ring with a silver border.
* Creates the inner blue and white quadrant checkerboard pattern.
* Adds the "B M W" text typography.
* Runs purely on Python's standard library (no external heavy dependencies required).

* `turtle` module (This comes pre-installed with standard Python installations, but Linux users may need to install `python3-tk`).
